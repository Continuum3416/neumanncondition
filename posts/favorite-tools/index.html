VS Code Extensions, IDEs, Softwares, Chrome Extensions

A list of tools that I use

Updated April, 11, 2024

Chrome Extensions

  • Viewport Resizer: Device Emulator & Website Testing Tool. I use this tool to test my websites on different viewports, simulating different devices. Chrome developer tool can do the same thing, but I like this one more.
  • Octotree: this tool let you preview files in a github repo. To view each individual folder on Github, I'll need to click on each of them to see all the files. However, Github is slow, so moving between different folders can take quiet some time.

VS Codes Extensions

  • Web Developments
    • HTML CSS Support
    • Auto Rename Tag.
    • CSS Peak
    • Tailwind CSS IntelliSense
    • Github Actions. I use this tool to automate some stuff for my website.
  • Decorations
    • indent-rainbow.
    • Material Icon Theme.
    • Prettier - Code formatter
    • Bracket Pair Colorization Toggler
  • System Programming
    • C/C++, C/C++ Extension Pack, and C/C++ Theme
    • CMake & CMake Tools.
    • Rust Analyzer.
    • MASM/TASM.
    • Nsight Visual Studio Code Edition
    • VS Code's Shader languages support: shaders highlighters for GLSL, HLSL, CG.
  • Others
    • audio-preview
    • vscode-pdf
    • Markdown All in One.

Softwares

  • Winhawk. Windows 11 sucks, as it does not allow you to customize even simple things like the size of the taskbar, so I used this software just to do that. It also has a lot of other cool tools that you can install on your machine.
