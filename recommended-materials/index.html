Recommended Materials

These are some resources that I think would serve as good learning materials.

Books

A Complete Guide to Programming in C++
by Ulla Kirch-Prinz and Peter Prinz
A good book for students to learn C++. This cover the language from the ground up, from basics to advanced.
You can buy this on: Amazon
Digital Image Processing
by Rafael Gonzalez and Richard Woods
A good introduction to theoretical computer vision. I like the detail description of point, line, edge detection and the SIFT algorithm of this book.
You can buy this on: Amazon
Data Structures and Algorithms in C++
by Michael T. Goodrich, Roberto Tamassia, and David M. Mount
A classic for C++ DSA.
You can buy this on: Amazon, Wiley
Introduction to Applied Partial Differential Equations
by John M. Davis
The first book I used to learn PDEs. Very good treatment of eigenvalue and boundary-value problems.
You can buy this on: Amazon, Abebooks
Applied Partial Differential Equations with Fourier Series and Boundary Value Problems
by Richard Haberman
A more detailed treatment of PDEs for the undergraduate level, covering higher-dimensional PDE's, nonhomogeneous problems, green functions, and method of finite difference.
You can buy this on: Amazon
Principles of Mathematical Analysis
by Walter Rudin
A standard, good university book for learning mathematical analysis over metric spaces. It's a good book if you have a dedicated instructor, not so good as a self-learn book.
You can buy this on: Amazon, Abebooks
Analysis I and II
by Terence Tao
An easier-to-understand analysis book series. The first book focuses on analysis on the real numbers and its subsets. Metric space is introduced in the second book.
You can buy this on: Amazon
Introduction to Classical Mechanics: With Problems and Solutions
by David Morin
An intuitive introduction to classical mechanics with interesting (difficult, even) problems.
You can buy this on: Amazon
Introduction to Electrodynamics
by David J. Griffiths
If you truly want to learn Electrodynamics, this book is the one, written by none other than David J. Griffiths himself. Problems are very hard, but interesting.
You can buy this on: Amazon, Ebay, Abebooks

Websites

Science and Mathematics Applied to Computer Science (SMACS): smacs.vercel.app. Shout out to my friend for creating this nice website for my school's SMACS club. As mentioned on the website's front page, SMACS aims to explore the intersections of science, math, and computer science. The website seeks to understand mathematical principles, scientific theories, and computation methods and apply them in computer science to simulate real life, solve calculations, or visualize abstract concepts.

beltoforion: beltoforion.de. This is one of the sites that inspired me. Developed by Ingo Berg, the website focuses on a wide range of topics, ranging from programming, earth and space science, mathematics, and physics.

Ten Minute Physics: link. Cool, complex physics simulation using only HTML and Javascript. This is a website created to accompany the youtube channel Ten Minute Physics.

scipython: scipython.com. One of my inspiration for this website. Created by the author of Learning Scientific Programming with Python, Christian Hill. This website contains an endless resource of Python simulation and scientific programming articles. This website is an overall excellent source of learning scientific programming.

Romanian Mathematical Magazine: ssmrmh.ro. Got bored by regular school mathematics? Want some extremely challenging problems that make you question your sanity? Come to this website.

cplusplus: cplusplus.com. A good website for learning C++ from start to end.

Mozilla Official Website for Developers: developer.mozilla.org. HTML, CSS, and especially Javascript.

W3 - CSS: w3schools.com. The best website for learning HTML and CSS.

Art of Problem Solving: artofproblemsolving.com. A forum-based website for math problems of all levels: middle school, high school, college, contests, olympiads, Putnam, etc.
