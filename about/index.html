About

More about this website (and me)

My name is Don, I'm currently a computer science student in Southern California. As part of my major and personal preferences, I spent a lot of time reading research papers and online scientific blogs, as a result, I got inspired by some of them and decided to create this website. This website was created for uploading my writings and projects. The topics that I write about range from mathematics, computer science, physics, and more. I'm also a math and computer science tutor. I left some of my personal notes in the Notes section above, feel free to use them.

I coded this website from scratch using HTML, CSS, vanilla Javascript, and jQuery (ancient tools, I know, but they works). To view the source code, click on the logo on the side navigation bar. Some other tools I used are MathJax, a javascript library that helps display fancy equation like this:

$$ \rho h\frac{\partial^2 w}{\partial t^2} = - D\nabla^4 w + T\nabla^2 w -K_0w -K_1\pd{w}{t} + T_1\nabla^2\pd{w}{t} + F(\textbf{x}, t), $$
and prism.js to provide some nice code syntax-highlighting for websites (you can change the theme in the setting).

About the logo: that's the trajectory of a double pendulum simulation. The general layout of this website should be completed, but I will still add some new features every once in a while, if I have time. This website is still relatively new anyway.
