Privacy Policy

This website does not collect or store any personally identifiable information, such as IP addresses. However, it does use Google Analytics to analyze website usage and user behavior. The data collected by Google Analytics includes:

  • Country of access: The geographical location from which users access the website.
  • Page title and screen class: The "Pages and Screens" report provides insights into how users interact with individual web pages or app screens, helping us (me) understand user engagement.
  • Traffic acquisition: The "Traffic Acquisition" report tracks where users are coming from, including the channels or sources that bring users to the site. This report focuses on session-based data, offering insights into session attributes such as the medium, source, and campaign.

Please note that the data gathered is used solely for analytical purposes and to improve the overall user experience on the website.
